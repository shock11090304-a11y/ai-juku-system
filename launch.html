🚨 よくあるトラブルと対処法

デプロイ後、ページが表示されない

1. Render Dashboard → Logs でエラー確認
2. 環境変数が全て設定されているか確認
3. render.yaml の buildCommand / startCommand を再確認

Stripe決済が通らない

1. Stripe Dashboard → Logs で失敗の理由を確認
2. テストモードと本番モードの切り替えを確認（sk_test_... vs sk_live_...）
3. Webhook署名シークレットが正しく設定されているか確認

Claude APIからのレスポンスが遅い/失敗する

1. console.anthropic.com で残高確認
2. Rate limit に達していないか確認
3. モデル名（claude-sonnet-4-6）がタイポしていないか確認

広告費を使っても登録が増えない

1. GA4 でファネル分析：LP→フォーム→送信 のどこで離脱しているか
2. 広告キーワードと LPメッセージが一致しているか
3. 無料体験のハードルを下げる（CC登録不要を強調）
4. ヒートマップツール（Microsoft Clarity 無料）導入で改善ポイント特定
