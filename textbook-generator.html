🐍 開発者向け: reportlab 連携ガイド（上級者のみ）

※ この欄は Python + reportlab 経験者向けの高度な使い方です。通常は上の「PDFで保存」ボタンで十分です。

1. 生成した Python配列をコピー

右上「🐍 Python配列」ボタンで CHAPTERS 形式の配列をコピー。

CHAPTERS = [
  {
    "title": "関係代名詞",
    "sections": [
      {"type": "intro", "text": "..."},
      {"type": "theory", "text": "..."},
      {"type": "example", "q": "...", "a": "..."},
      ...
    ]
  }
]

2. 既存の .py ファイルに貼り付け

eiken_pre1_textbook.py / kobun_textbook.py の CHAPTERS 変数に上書き。

# 上書き例
CHAPTERS = [ ... ]  # 生成したコンテンツ

# 既存のビルド関数が自動で処理
build_pdf(CHAPTERS, output_path='output.pdf')

3. PDF ビルド

ターミナルで実行するだけ。

python3 eiken_pre1_textbook.py
# → textbook.pdf が生成される
