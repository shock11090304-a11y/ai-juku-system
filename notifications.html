📲 LINE Messaging API を本番稼働させるには

このテンプレートは既に完成しています。LINE Developers で Messaging API を有効化し、バックエンドに以下のエンドポイントを実装するだけで稼働します：

  1. /webhook - LINEからのメッセージ受信
  2. /notify/weekly - 毎週日曜20時に保護者へ週次レポート配信（Cron）
  3. /notify/streak - 未学習日の21時にリマインダー
  4. /notify/achievement - アチーブメント獲得時に即時配信

💡 バックエンド構築は次スプリント予定。Node.js + LINE SDK or Python + FastAPI で2-3日で構築可能。
