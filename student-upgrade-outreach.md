## 📊 想定される質問と回答

### Q. 「AIだけで十分じゃないですか？」

```
A. いい質問です。実は、AIだけに頼る学習は
落とし穴があります。AIは間違いを指摘されても、
その場で気付けません。

当塾のプランは「AI + 講師の毎週レビュー」で、
AIの誤解や行き詰まりを講師が人間の目で
チェックする仕組みです。ここが外部の¥39,800の
契約にはない最大の価値です。
```

### Q. 「¥15,000は高くないですか？」

```
A. 個別指導の相場は1時間¥5,000〜¥8,000です。
AIは24時間質問し放題なので、週に3時間利用
するだけで月¥60,000相当の価値があります。

さらに、外部契約の¥39,800と比べると、
60%以上お得な設定になっています。
```

### Q. 「子供がAIに頼りすぎませんか？」

```
A. その懸念はもっともです。だからこそ
「AIに質問した履歴を講師が毎週確認する」
仕組みにしています。

AIに答えを聞くだけで終わっている生徒には、
次回授業で同じ問題を自分で解かせて、
本当に理解しているか確認します。
```

### Q. 「解約はいつでもできますか？」

```
A. はい、月末までにお伝えいただければ
翌月から停止できます。違約金は一切ありません。

ただし、AIの学習履歴は解約と同時に削除される
ため、再契約時に過去のデータは引き継げない
点だけご了承ください。
```
